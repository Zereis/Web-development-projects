Vad är CSS?

CSS står för Cascading Style Sheets och är en av webbens tre grundläggande byggstenar tillsammans med HTML och Javascript. Man kan säga att CSS är webbens ansikte utåt, och sköter alltså allt som har med design att göra.

Vilka Versioner finns det?

Det finns just nu CSS1, CSS2, CSS3 och CSS4 vilket är den nyaste versionen av CSS.

Varför finns det olika versioner? Varför finns det inte bara en?

Det finns fler versioner av CSS och varje ny version byggs på föregående version och gav nya möjligheter. Just nu så finns det 4 versionern av CSS. Olika versioner har deras fördelar och nackdelar. Tex CSS1 är den första css version som byggs och är en viktig del att utgå CSS mallenm ifrån. CSS2 Är rekommenderat just nu för folk att använda CSS, och CSS 3 som är uppdelad i mindre modular håller på att gå från den moderna implmenetation att bli normaliserd som rekommenderat.

Varför är det viktig att veta vilken version som en viss CSS egenskap tillhör?

Eftersom olika webbläsare tolkar kod på olika sätt så är det viktig att använda rätt version för att kunna arbeta.
